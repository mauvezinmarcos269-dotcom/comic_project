# 📊 美漫大数据智能问数与高级分析平台

本项目是《高级Python程序设计》2026年春季学期的期末项目。系统集成了数据采集、预处理清洗、PCA降维聚类分析以及基于自然语言处理的交互式智能问答系统。

## 🌟 项目简介
本项目聚焦于美国漫画工业的历史销售数据。通过构建自动化数据增强管线（Pipeline），对接 ComicVine API 获取元数据，对基础表单进行了深度的维度富化（如创作者、奖项、角色提取、连载生命周期判断）。最终在前端实现了多维度、交互式的可视化大屏展示。

### 核心功能模块
1. **智能数据管线**：支持断点续传、并发调用，并融合了本地正则兜底策略提取 `Publication_Status` 与核心角色。
2. **多维统计分析**：
   - 使用 `K-Means` 与 `PCA` 探寻数据潜在群落分布。
   - OLS 回归分析挖掘价格与销量的弹性关系。
3. **可视化看板**：利用 Plotly 绘制动态热力图、大盘走势图及厂商对决视图。
4. **Agent 智能问数**：基于文本特征正则匹配的问答解析器，支持模糊语义调用分析报表。

## ⚙️ 技术栈
- **语言**: Python 3.10+
- **数据处理**: Pandas, NumPy
- **机器学习/NLP**: Scikit-Learn, Statsmodels, re
- **可视化/前端**: Streamlit, Plotly
- **网络与并发**: Requests, concurrent.futures
